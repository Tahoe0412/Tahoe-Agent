# Econometrics Assignment 2B Bundle

This folder collects the econometrics materials we worked on in one place without moving or deleting the originals elsewhere in the workspace.

## Structure

- `inputs/`
  - original assignment PDF
  - reference paper PDF
  - `fredgraph.dta`
  - previous-year presentation template
- `analysis/`
  - R analysis script
  - Stata do-file
  - bilingual submission draft
- `results/`
  - plot PNG
  - AR model selection table
  - ADL model selection table
  - Stata batch log
- `presentation/`
  - current presentation workspace
  - copied source/assets and package files only
  - `node_modules` intentionally not copied to keep this bundle clean

## Notes

- All source files remain in their original locations.
- This bundle is a clean copy for assignment organization and delivery.
